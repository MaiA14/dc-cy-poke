# Pokeomon API

## Features
- Authentication:
    1) Register new user
    2) Login 
- Pokemon CRUD
- Bonus:
    1) Validations
    2) A search functionality to filter PokÃ©mon by name or type.

## Dependencies
- Docker

## Running this server 

```bash
$ docker-compose up
```

> [!IMPORTANT]
> You can change configuration (port, etc using env file)
